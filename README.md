# 貝達國際貿易有限公司網站

## 專案簡介

貝達國際貿易有限公司的官方網站，提供專業的國際貿易服務，包括採購代理、進出口服務、供應商媒合、報價與樣品安排、訂單協調等全方位解決方案。

## 網站特色

- 🌍 **多語言支援**: 中文、英文、日文三種語言切換
- 📱 **響應式設計**: 完美適配桌面、平板、手機各種設備
- ⚡ **現代化技術**: 純HTML/CSS/JavaScript，無框架依賴
- 🎨 **簡約設計**: 符合國際企業標準的專業形象
- 🚀 **優化性能**: 快速載入，流暢動畫效果
- 📧 **聯絡表單**: 完整的客戶聯絡和諮詢功能

## 使用說明

### 🚀 本地開發
1. 下載專案文件到本地
2. 使用現代瀏覽器開啟 `index.html`
3. 或使用本地伺服器：
   ```bash
   # 使用Python
   python -m http.server 8000
   
   # 使用Node.js
   npx serve .
   
   # 使用PHP
   php -S localhost:8000
   ```

### 📝 內容編輯
- **文字內容**: 直接編輯HTML文件中的文字
- **多語言**: 編輯 `assets/js/language.js` 中的翻譯對象
- **樣式調整**: 修改 `assets/css/` 中的CSS文件
- **圖片替換**: 更換 `assets/images/` 中的圖片文件

### 🌍 部署上線
1. 上傳所有文件到網頁伺服器
2. 確保支援HTTPS（推薦）
3. 配置域名指向網站根目錄
4. 測試所有功能和連結
